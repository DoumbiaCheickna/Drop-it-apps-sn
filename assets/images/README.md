# 📸 Guide des Images

## Structure des Dossiers

```
assets/images/
├── logos/              # Logos des applications
│   ├── dropit-logo.png
│   ├── dropit-pro-logo.png
│   └── dropit-trade-logo.png
│
├── icons/              # Icônes diverses
├── screenshots/        # Captures d'écran
└── backgrounds/        # Images de fond
```

## Spécifications des Logos

### Logos des Applications

Placez vos logos dans le dossier `logos/` avec ces noms :

1. **dropit-logo.png** (DROP IT)
   - Format : PNG avec transparence
   - Taille recommandée : 200x200px
   - Ratio : 1:1 (carré)

2. **dropit-pro-logo.png** (DROP IT PRO)
   - Format : PNG avec transparence
   - Taille recommandée : 200x200px
   - Ratio : 1:1 (carré)

3. **dropit-trade-logo.png** (DROP TRADE)
   - Format : PNG avec transparence
   - Taille recommandée : 200x200px
   - Ratio : 1:1 (carré)

### Logo Principal (Header)

4. **logo-main.png**
   - Format : PNG ou SVG
   - Taille : 48x48px (ou vectoriel)
   - Pour le header du site

## Formats Acceptés

- **PNG** : Recommandé pour les logos avec transparence
- **SVG** : Idéal pour les icônes vectorielles
- **WebP** : Alternative moderne pour meilleure compression
- **JPG** : Pour les photos uniquement

## Optimisation

Avant d'utiliser vos images :
1. Compressez-les avec [TinyPNG](https://tinypng.com)
2. Utilisez des formats modernes (WebP)
3. Gardez la transparence pour les logos
4. Exportez en @2x pour les écrans Retina

## Où Placer Vos Logos ?

Copiez simplement vos fichiers dans :
```
Drop-it/assets/images/logos/
```

Les logos seront automatiquement affichés sur le site !
