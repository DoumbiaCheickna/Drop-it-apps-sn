## 📝 Contribuer à la Documentation

La documentation peut toujours être améliorée ! Pour contribuer :

1. Fork le repository
2. Créez une branche pour votre documentation
   ```bash
   git checkout -b docs/amelioration-guide
   ```
3. Ajoutez ou modifiez la documentation
4. Créez une Pull Request

### Conventions d'Écriture

- Utilisez le Markdown
- Ajoutez des exemples de code
- Incluez des captures d'écran si pertinent
- Restez clair et concis
- Utilisez des émojis pour la lisibilité 😊
